# dockercompose-java-html
docker compose

**You can compile it using:**
javac HelloWorld.java

/project-directory
  ├── HelloWorld.java
  ├── HelloWorld.class
  ├── Dockerfile.java
  ├── Dockerfile.html
  ├── index.html
  └── docker-compose.yml

 ** Build and Run**
**To build and run the containers, use the following commands:**

docker-compose build
docker-compose up
